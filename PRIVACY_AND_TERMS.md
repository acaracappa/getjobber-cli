## Privacy Policy

### Data Collection

**getjobber-cli does NOT collect, store, transmit, or share any user data or analytics.**

- **No Telemetry**: This tool does not send any usage statistics, error reports, or telemetry data to any external servers.
- **No Analytics**: We do not track how you use this tool, what commands you run, or what data you access.
- **No Personal Information**: We do not collect names, email addresses, or any other personal information.
- **Local Storage Only**: All configuration data (OAuth credentials, tokens) is stored locally on your device using OS-level secure storage (Keychain/Credential Manager) or encrypted local files.

### Third-Party Services

This tool interacts with the GetJobber API on your behalf using your OAuth credentials. Any data accessed through the GetJobber API is subject to [GetJobber's Privacy Policy](https://getjobber.com/privacy/) and [Terms of Service](https://getjobber.com/about/terms/).

### Data Security

- OAuth tokens are stored securely using your operating system's native credential management system (macOS Keychain, Windows Credential Manager, Linux Secret Service)
- Configuration files are created with restricted permissions (0600/0700 - readable only by the file owner)
- All API communications use HTTPS/TLS encryption
